# Project Agents Notes

- Maintain a root-level `progress.md` file for this project.
- After every meaningful project update, append a one-line entry under the current date heading in `progress.md`.
- Use a `## YYYY-MM-DD` date heading for each day and flat `- ` bullet items beneath it.
- Keep entries concise and outcome-focused.
- Record only what changed, not how it was implemented.
- Avoid code snippets, implementation details, and long explanations in `progress.md`.
